About Bloggato

Bloggato (which name derives from an italian term that means "written on a blog") is a very simple blog written using Python/Django and hosted on Google App Engine. In particular django-nonrel fork has been used, because GAE doesn't support relational databases. In the html templates Bootstrap has been used to make the UI nicer and ready to be viewed on a mobile device.

Features are very simple and basic: it's possible to insert a new post specifying title and content (you need to have an account already created) and users (logged in or anonymous ones) can leave a comment on each post. Only the author of a post can delete it and when an anonymous user tries to perform an action reserved to registered users, he/she is automatically redirected to the login page.

Bloggato is released under the MIT license and all the source code is available at https://github.com/andreagrandi/bloggato
